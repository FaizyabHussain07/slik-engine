## ⚡ The Problem

Building a modern SaaS app currently requires assembling disjointed tools and generic boilerplates. You usually spend hours (or days) configuring Next.js, wiring up Tailwind CSS, picking a UI library, writing auth routes, and wiring the database, only to be met with a barebones design.

## 🚀 The Slik Solution

Slik completely skips the boilerplate phase. Running a single command bootstraps a complete project that not only functions perfectly under modern architectures but looks **handcrafted and premium** right out of the box. Focus squarely on your unique business logic instead of rewriting standard UI components.

## 📦 Getting Started

Start your new project instantly across terminal:

```bash
npx create-slik@latest my-app
```

You'll quickly be prompted to select your configurations:
- **Stack:** Next.js / React
- **Themes:** Bento / Frost / Mono (Strictly beautiful palettes and glassmorphism)
- **Database:** PostgreSQL via Prisma included (Optional)
- **Auth:** NextAuth/Auth.js completely mapped out

Once done:
```bash
cd my-app
npm install
npm run dev
```
Open `http://localhost:3000` and see your premium template live!

## 💎 Features

- **Blazing Fast Init**: No endless configuration prompts. Get started in precisely 60 seconds.
- **Premium Handcrafted UI**: Bypass clunky raw components. The engine injects fully animated (Framer Motion), highly aesthetic layouts and components instantly utilizing Tailwind CSS.
- **Dark Mode by Default**: Deep blacks, subtle glass borders, and smooth 0.3s transitions everywhere. Completely localStorage persistent. 
- **Database & Auth Hooks Included**: Schema setup and OAuth integrations pre-wired.
- **Responsive Contexts**: Mobile navigation structures, layout drawers, and grids ready for any scale.

## 🛠 Tech Stack Produced

When you use Slik to generate a project, your new app will natively rely on:
- **Framework**: React / Next.js
- **Foundations**: TypeScript
- **Styling**: Tailwind CSS + `clsx` & `tailwind-merge`
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Backend Infrastructure**: Supabase & PostgreSQL & Prisma (If opted)
