# Installation

## Requirements

| Requirement | Version |
|-------------|---------|
| Node.js | >= 18.0.0 |
| npm / pnpm / bun | Latest |
| Git | Any recent version |

## Using npx (Recommended)

No installation required:
```bash
npx create-slik@latest
```

## Using npm

```bash
npm create slik@latest
```

## Using pnpm

```bash
pnpm create slik@latest
```

## Using bun

```bash
bunx create-slik@latest
```

<Note>
  We recommend using npx so you always get the latest version.
</Note>
