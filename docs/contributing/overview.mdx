# Contributing

Slik is open-source and welcomes contributions.

## Ways to Contribute

- ⭐ Star the repo
- 🐛 Report bugs
- 💡 Suggest features
- 🎨 Submit new design vibes
- 📝 Improve documentation
- 🔧 Fix bugs or add features

## Development Setup

```bash
git clone https://github.com/FaizyabHussain07/slik-engine.git
cd slik-engine
pnpm install
pnpm dev:cli
```

## Project Structure

## Submitting a PR

1. Fork the repo
2. Create a branch: `git checkout -b feature/my-feature` 
3. Commit: `git commit -m "feat: add my feature"` 
4. Push and open a Pull Request
