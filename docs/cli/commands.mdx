# CLI Commands

## create-slik

The main command to create a new Slik project.

```bash
npx create-slik@latest [project-name]
```

### Options

Currently, all options are interactive. Non-interactive mode coming soon.

### Examples

```bash
# Interactive mode (recommended)
npx create-slik@latest

# With project name (still interactive for stack/vibe)
npx create-slik@latest my-app
```
