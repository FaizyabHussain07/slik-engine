# Slik

**Design at the speed of thought.**

Slik is an open-source CLI tool that generates beautiful, production-ready full-stack SaaS applications in under 60 seconds.

<Note>
  Slik is free forever. Open-source under MIT License.
</Note>

## The Problem

Every new project starts the same way:
- Set up authentication
- Build a landing page
- Create a dashboard
- Configure the database
- Wire up email and OAuth
- Style everything from scratch

This takes **hours** — before you write a single line of your actual product.

## The Solution

```bash
npx create-slik@latest
```

One command. Three questions. A complete, production-ready project in under 60 seconds.
