# Quick Start

<Steps>
  <Step title="Run the CLI">
    Open your terminal and run:
```bash
    npx create-slik@latest
```
    No installation needed. npx downloads and runs Slik automatically.
  </Step>

  <Step title="Answer 3 questions">
    The CLI will ask you:
✦ slik — design at the speed of thought

◆  Project name?
   › my-app

◆  Pick a stack?
   ● Next.js 15     SSR · App Router · Recommended
   ○ React (Vite)   SPA · Client-side only
   ○ Vanilla HTML   Zero framework · Ultra-light

◆  Pick a vibe?
   ● Bento   Grid-based · Dark · Editorial
   ○ Frost   Glassmorphism · Light + blur
   ○ Mono    Minimal · Monochrome
  </Step>

  <Step title="Set up Supabase">
    1. Go to [supabase.com](https://supabase.com) and create a free project
    2. Go to **SQL Editor** and run the migration:
```bash
    # File is inside your project
    supabase/migrations/001_initial.sql
```
    3. Go to **Auth → Providers → Google** and enable Google OAuth
  </Step>

  <Step title="Fill environment variables">
    Open `.env.local` in your project root:
```env
    NEXT_PUBLIC_SUPABASE_URL=your_project_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
    SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
    NEXT_PUBLIC_SITE_URL=http://localhost:3000
```
    Find these values in: Supabase Dashboard → Project Settings → API
  </Step>

  <Step title="Start the dev server">
```bash
    cd my-app
    npm run dev
```
    Open [http://localhost:3000](http://localhost:3000) — your app is running! 🎉
  </Step>
</Steps>

<Check>
  Done! You now have a complete full-stack SaaS with landing page, auth, dashboard, and admin panel.
</Check>
